# 1.排序

## 1.1冒泡排序

BubbleSort

时间复杂度O(N*N)，额外空间复杂度O(1)

## 1.2选择排序

SelectionSort

时间复杂度O(N*N)，额外空间复杂度O(1)

## 1.3插入排序

InsertionSort

时间复杂度和数据的原始状况有关：

    1.最理想的状态原始数据就是按照从小到大排序的，这时时间复杂度为O(N);
    2.最差的状态是原始数据按照从大到小排序，这时时间复杂度度为O(N*N).
额外空间复杂度O(1)
